Mac os创建React项目（windows同）

2019-11-13 React React 0 评论

1.下载node.js。

略。

2.安装cnpm。（可以略过，使用国内的下载速度会好些）

终端：npm install -g cnpm --registry=https://registry.npm.taobao.org

3.全局安装。(2选1)

终端：npm install -g create-react-app（安装国外的镜像）
终端：cnpm install -g create-react-app（安装国内的镜像）

4.进入自己的工作空间。（平时项目放置的文件）

终端：cd xxx/xxx/（进入自己的项目文件）
终端：create-react-app 项目名（创建自己的React项目）
终端：npm init（初始化）

5.导入react和react-dom。

终端：npm install --save react react-dom

6.安装路由。

终端：npm install --save react-router-dom

7.启动。

终端：npm start
