Mac os配置Tomcat

2019/06/24 (发布) 笔记 Tomcat 0 评论 字数: 160(字)阅读时长: 1(分)

温馨提示: 点击下方 [ 文章目录 ] 可以看文章目录，祝阅读愉快，不解之处请积极联系博主。

一、 下载

登录Apache Tomcat官网，地址http://tomcat.apache.org点击左边的Download，选择需要下载的版本。

二、设置本地放置路径

把下载下来的包解压到 /Users/你的用户名/目录下

三、启动Tomcat

打开终端
cd /Users/你的用户名/apache-tomcat-9.0.0.M21/bin，回车
赋予超级管理员权限sudo chmod 755 *.sh，
输入：./startup.sh，就可以启动我们自己的Tomcat了
打开我们的浏览器，输入 http://localhost:8080，如果出现一只猫，则证明配置成功。

四、关闭Tomcat

同样是在bin目录下，在终端输入：./shutdown.sh+ 回车，就可以了。
