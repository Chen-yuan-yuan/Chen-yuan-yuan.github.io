谷歌浏览器的控制台

  • Elements：查看结构样式。
  • Console：查看输入和报错。
  • Sources：查看源码。
  • Network：查看所有的资源的请求信息，加载时间等。（性能调优）
  • Application：查看网站的数据存储和资源文件。（盗图）

JS做客户端语言

按照js的相关语法，操作页面的元素，有时还要去操作浏览器的一些功能。

  • ECMAScript3/5/6…:JS的语法规范。（变量，数据类型，操作语句等）
  • DOM（document object model）：文档对象，提供一些JS的属性和方法，用来操作页面的DOM元素
  • BOM（browser object model）：浏览器对象模型，提供一些JS的属性和方法，用来操作浏览器的。

变量：可变的量。

JS中的命名规范

  • 严格区分大小写。
  • 使用数字、字母、下划线。
  • $、数字不能作为开头。
  • 公共变量用"_"开头。
  • 使用驼峰命名法（首字母小写）
  • 不能使用关键字和保留字

常用缩写

  • add、insert、create、new（新增）
  • delete、del、remove、rm（删除）
  • sel、select、query、get、info（查询）
  • update、（更新）

number数字类型

包含：常规数字类型、NaN

NaN

not a number:不是一个数，但属于数字类型

NaN和任何值（包括自己）都不相等。

isNaN(“xxx”)

检测一个值是否为非有效数字。
返回：不是有效数字返回true，是有效数字返回false。
原理：使用isNaN()进行检测都时候，1.会先检测是否为数字类型，先基于Number()方法，把值转换为数字类型，2.然后再检测。
