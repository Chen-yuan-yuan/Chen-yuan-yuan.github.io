Mac os搭建Angular项目

2020-06-24 Angular 0 Comments

略。

略。

cnpm install -g @angular/cli

创建一个Angular项目。

ng new 你的Angular项目名称

进入工作空间。（就是上一步的项目文件夹）

cd xxx/xxx

启动。

ng serve --open
