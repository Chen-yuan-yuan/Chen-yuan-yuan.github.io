Mac 配置Vue环境

2020-06-24 Vue Comments

1.检查node版本

node -v

2.查看npm版本

npm -v

3.安装vue脚手架

sudo npm install -g vue-cli（安装的是vue-cli2.x）

sudo npm install -g @vue/cli （安装的是vue-cli3.x）

出现错误的处理：

清除缓存的命令：npm cache clean --force

清除缓存后，用vue -V 命令查看版本

4.进入自己的项目目录

cd xxx(项目目录)

5、进入工作空间

命令：cd xxx(工作空间)

6、创建项目

命令：vue create 项目名

7、选择手动配置组件或者默认

略。（网上很多，可以自己找）

8、启动。

npm start
