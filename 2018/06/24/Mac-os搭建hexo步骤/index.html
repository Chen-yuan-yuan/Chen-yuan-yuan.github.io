Mac os搭建hexo步骤

2018-06-24 Hexo Comments

1.获取管理员权限。

sudo su

2.安装cnpm。

npm install -g cnpm --registry=https://registry.npm.taobao.org

3.安装hexo。

cnpm install -g hexo-cli

4.查看hexo版本。

hexo -v

5.查看安装的路径。

pwd

6.建立一个名为“blog”的文件夹。

mkdir blog/

7.进入"blog"的文件路径。

cd blog

8.安装hexo。

sudo hexo init

9.启动hexo博客。

hexo s

10.退出。

Ctrl+C

11.新建一篇文章。

hexo n “新建文章的名称”

12.查看效果。

hexo clean（刷新）
hexo g（生成文件）
hexo s（启动）
